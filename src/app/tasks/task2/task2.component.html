Задача 02
Разобьем ее на несколько частей, будешь сдавать ее частями.

Часть первая
Добавить по клику на кнопку "add random number" случайное число от 1 до 100 в коллекцию внизу, там где сейчас 30, 3, 6. По клику число должно добавляться в конец списка

Часть вторая
Вывести справа статистику по введенным числам:
  • Количество введенных чисел
  • минимальное значение из всех чисел
  • максимальное значение из всех чисел
  • среднее значение из всех чисел
Numbers in collection:
0, 20, 30, 40
Statistics: (вторая часть задачи)
Count: 0
Min: 1
Max: 2
